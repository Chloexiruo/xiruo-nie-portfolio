# Xiruo Nie - Data Portfolio

A responsive personal portfolio for postgraduate programme and internship applications,
with a focus on mathematics, statistics, data analysis, quantitative finance, risk, and
research.

## Highlights

- Academic background and relevant areas of study
- GPU software and Python internship experience
- DataFest, mathematical modelling, and time series projects
- Awards, quantitative skills, and downloadable CV
- Accessible SVG project visualisations
- Responsive desktop and mobile layouts

## Technology

- React
- Vite
- JavaScript
- CSS
- Inline SVG visualisations

## Run locally

```bash
npm install
npm run dev
```

Create a production build with:

```bash
npm run build
```
